# Student Record System

A simple Java OOP project demonstrating encapsulation, classes, and objects.

## Features
- Add new students
- Display all students
- Search student by roll number

## OOP Concepts Used
- **Encapsulation**: Private fields with public getters
- **Classes and Objects**: Student and School classes
- **ArrayList**: Dynamic collection of students

## How to Run

### On Linux/Mac:
chmod +x run.sh
./run.sh

### On Windows:
javac src/*.java
java -cp src Main
